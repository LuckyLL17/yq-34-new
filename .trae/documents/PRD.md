## 1. 产品概述

字帖生成器是一款面向书法爱好者、学生和教育工作者的在线工具，支持生成数字、汉字和英文三种类型的字帖，可切换多种书法字体，并提供 PDF 导出功能供打印练习使用。

- 解决用户快速创建个性化练字帖的需求，无需专业设计软件
- 目标用户：书法初学者、学生家长、语文/英语教师、书法爱好者

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：字帖配置区、实时预览区、操作按钮区

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 文字类型选择 | 切换数字/汉字/英文字帖类型，每种类型提供预设内容 |
| 主页面 | 文字输入区 | 支持自定义输入文字内容 |
| 主页面 | 字体选择器 | 下拉选择不同书法字体（楷体、行楷、宋体、草书、隶书、英文手写体等） |
| 主页面 | 字帖样式配置 | 配置网格类型（田字格、米字格、回宫格、无格线）、格子大小、每行字数、行数 |
| 主页面 | 颜色配置 | 配置字体颜色、网格颜色、是否显示参考虚线 |
| 主页面 | 实时预览区 | 所见即所得的字帖预览，支持滚动查看 |
| 主页面 | PDF 导出 | 一键将字帖导出为 PDF 文件，支持 A4 纸张配置 |

## 3. 核心流程

用户进入页面 → 选择字帖类型 → 输入或选择预设文字 → 选择字体 → 调整字帖样式参数 → 实时预览效果 → 点击导出 PDF → 下载文件

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：米白色背景 (#FAF7F2) 搭配深褐色文字 (#3D2C1F)，营造宣纸古风质感
- **辅助色**：墨红色 (#8B2E20) 用于强调按钮和重要提示
- **按钮风格**：圆角矩形按钮，悬浮时微抬升效果，墨红按钮用于主操作
- **字体**：标题使用"Noto Serif SC"宋体，正文使用系统字体，预览区根据选择动态切换
- **布局风格**：左右分栏布局，左侧配置面板，右侧大面积预览区
- **图标风格**：使用线性 Lucide 图标，保持简洁雅致

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主页面 | 顶部标题区 | 书法体 Logo、副标题、轻微宣纸纹理背景 |
| 主页面 | 左侧配置面板 | 卡片式分组布局，每组配标题和折叠功能，下拉选择器和滑块控件 |
| 主页面 | 右侧预览区 | 带阴影的纸张模拟效果，田字格/米字格背景，可滚动容器 |
| 主页面 | 底部操作区 | 固定悬浮的导出按钮组，重置配置按钮 |

### 4.3 响应式

- 桌面端：左右分栏（配置区 380px，预览区自适应）
- 平板端：上下分栏，配置区可折叠
- 移动端：单列布局，预览区全屏，配置区通过底部抽屉展开
- 触摸优化：滑块和按钮增加点击热区
